# Dimension 05.07: Memory Privacy, Scope, and Deletion

## Purpose

Study how memory handles sensitive information and user boundaries.

## Steps

1. Inspect memory scopes.
2. Search for privacy filters.
3. Check retention policies.
4. Inspect deletion/anonymization.
5. Check whether memories are encrypted or access-controlled.
6. Check whether memory writes are audited.

## Evidence to Capture

- Privacy filters
- Retention config
- Delete APIs
- Tenant filters
- Encryption config
- Access checks

## Questions

1. Can memory leak between users?
2. Can users delete memory?
3. Is sensitive data stored?
4. Is memory access audited?
5. Are scopes enforced in queries?

# Dimension 15.02: Message Routing and Termination

## Purpose

Study routing rules, speaker selection, handoff semantics, group termination, and deadlock prevention.

## Steps

1. Identify message routing logic.
2. Check speaker selection strategy.
3. Inspect handoff semantics and contracts.
4. Check group termination conditions.
5. Look for deadlock prevention mechanisms.

## Evidence to Capture

- Router implementations
- Speaker selection code
- Handoff handlers
- Group termination conditions
- Deadlock detectors

## Questions

1. How are messages routed?
2. How is the next speaker selected?
3. How are handoffs managed?
4. When does a group conversation terminate?
5. Is deadlock possible?

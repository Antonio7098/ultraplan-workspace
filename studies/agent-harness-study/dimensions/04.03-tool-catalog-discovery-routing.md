# Dimension 04.03: Tool Catalog, Discovery, and Routing

## Purpose

Study how the system decides which tools are available and relevant.

## Steps

1. Identify global vs per-agent tool lists.
2. Inspect dynamic tool filtering.
3. Check whether tools are selected by capability, permission, or context.
4. Identify whether retrieval is used for tool discovery.
5. Check whether tool availability is traced.

## Evidence to Capture

- Tool catalogs
- Capability filters
- Agent configs
- Tool retrieval
- Routing policies
- Dynamic availability code

## Questions

1. Does every agent see every tool?
2. Are tools filtered by task?
3. Are tools filtered by permission?
4. Can tools be hidden from the model?
5. Is tool availability explainable?

## Rating

| Score | Meaning |
|-------|---------|
